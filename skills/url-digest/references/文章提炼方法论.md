# 公众号文章提炼模板

## 一、元信息
```json
{ "title": "", "link": "", "author": "", "source": "", "publish_date": "" }
```

## 二、是否软广
```json
{ "is_ad": true/false, "reason": "判断依据", "score": 1-5 }
```

## 三、作者身份
```json
{ "job": "职业", "industry": "行业", "influence": "影响力" }
```

## 四、主题与实体
```json
{ "themes": ["主题1"], "key_entities": ["实体1"] }
```

## 五、读者群体
```json
[{ "行业": "互联网", "职业": "产品/研发", "性别": "不限", "年龄": "25-45岁" }]
```

## 六、文章价值
- 定性：解决什么问题、发现什么趋势、创造什么工具
- 定量：影响范围、人群量级、频率、价值程度
```json
{ "value_analysis": [], "impact_scopes": "", "frequency": "高/中/低", "value_uplifts": "高/中/低", "score": 1-5 }
```

## 七、核心观点与金句
金句需注明：谁说的+时间+来源
```json
{ "quote": [], "core_insights": [] }  // 各最多5个
```

## 八、关键速览
每个段落的核心观点/洞察，最多10条
```json
{ "highlights": [] }
```

## 九、内容标签
```json
{ "tags": ["tag1"] }  // 最多5个
